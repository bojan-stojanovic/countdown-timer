# Countdown timer  
This is simple countdown timer, you just simply need to set end date.  

## Setting end date
You can set end data in `data-count-down-to` html attribute.  

## Date format
Date must be in `YYYY-MM-DD` format.  

## Date example
`data-count-down-to="2023-12-31"`
